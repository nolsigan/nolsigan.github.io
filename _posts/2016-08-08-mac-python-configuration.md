Mac에서 Python 환경설정 할 때마다 까먹어서 암걸린다..

적어놔서 다음부턴 보고서 해야지 ㅠㅠ



1. pip-2.7 or python2.7 install pip 이용하기
2. Mac의 default python을 사용하면 안된다. 경로는 /usr/local/lib
3. easy_install, pip를 바로 가져다쓰면 default 경로에 다운된다.
2. MacPorts를 쓰고 있으므로 pip-2.7, python2.7 써서 /opt/local/Library에 설치되도록 한다.
